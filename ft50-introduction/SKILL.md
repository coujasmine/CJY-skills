# Part: Introduction

## 目标
把“变量题”改写成“现象张力题”，并明确理论世界观归属。

## 执行步骤
1. 检查是否命中创新/创业核心要素：不确定性、新颖性、机会、搜索、涌现、价值创造/配置。
2. 若是 X→Y 变量表达，先改成 1-2 句现象级问题。
3. 标注世界观传统（机会/动态能力/商业模式/社会创业/涌现/认知）。
4. 给出 FT50 放大路径（现象层级、理论争论、制度或技术背景）。

## 写作模板
- 段1：现实张力与关键现象（非“文献空缺”）
- 段2：现有解释为何不足（指出具体理论失灵点）
- 段3：本文问题与理论定位（主干理论+张力）
- 段4：贡献预告（机制、边界、情境）

## 红线
- 出现“很少研究X对Y影响”作为核心动机：判不合格。
- 仅更换创业样本而无机会/涌现逻辑：判不合格。
